github使用小乌龟的ssh

1.在github上创建一个仓库点击绿色的按钮创建仓库

2、创建成功的仓库如图

3，在本地生成key，key有两个，一个是publickey放在github上的，第二个的是privatekey，是放在本地的。安装TORTOISEGIT就不说了，找到安装目录，找到生成key的工具，puttygen.exe点击打开

4.打开如图

5.点击generate后，鼠标要在空白地方随意移动，后生成key，上面框中的是publickey

6.生成key后，回到github上。先选择settings 再点击左边Deploykeys，点击add deploykey 。然后把title填好 key填入上一步生成的publickey，即是上面在框框里生成的key，填入后。记得选择allow write access 。否则这个key只有只读权限。选上则有可写权限。然后点击绿色的addkey即可

7.再回到生成key的软件上，点击save private key 把key保存到本地

8.选个文件夹，点击右键。点开tortoisegit的settings，将一些配置，配置好，比如name和email，点开settings的network 注意ssh的配置，必须是tortoisegit的安装目录下的tortoiseplink.exe文件

9.配置完后。随便找个空文件夹，右键呼出git，然后点击clone将github上的项目拉下来。记得选择load putty key 这里选择刚才保存的privatekey 如图

10.然后就可以看到拉取下来的项目，如图

11.点击项目。右键，选择settings。点开git remote 。如图。配置一下，remote为名字，随便取一个，URL 填入仓库的ssh地址 KEY 和URL一样，putty key 选择保存的privatekey 点击应用

12.在项目里随便建个文件，comit到本地，再push到github上。点击push后弹出来的对话框。注意remote选择上一步设置的remote 下面记住选择autoload puttykey。点击ok。即可上传到github上
